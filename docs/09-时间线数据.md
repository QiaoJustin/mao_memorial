## 四、数据导入说明

### 4.1 导入方式

1. **数据库导入**：将上述数据通过 Prisma Seed 脚本导入数据库
2. **JSON 文件**：将数据保存为 `src/data/timeline.json`，前端直接引用
3. **API 导入**：通过后台管理 API 逐条录入

### 4.2 图片处理建议

1. **本地化存储**：建议将图片下载到本地服务器或 OSS，避免依赖外部链接
2. **缩略图生成**：使用 sharp 库生成多尺寸缩略图（200px / 600px / 1200px）
3. **格式转换**：建议转换为 WebP 格式，减小文件体积
4. **CDN 加速**：生产环境建议使用 CDN 加速图片访问
