## 1. 项目背景

### 1.1 背景

毛泽东同志（1893年12月26日—1976年9月9日）是伟大的马克思主义者，伟大的无产阶级革命家、战略家、理论家，是马克思主义中国化的伟大开拓者，是近代以来中国伟大的爱国者和民族英雄，是党的第一代中央领导集体的核心，是领导中国人民彻底改变自己命运和国家面貌的一代伟人。

毛泽东同志的一生，是为中华民族的独立、中国人民的解放和中国社会的进步不懈奋斗的一生。从青年时代求学探索真理，到领导秋收起义、开辟井冈山革命根据地；从万里长征、延安抗战，到指挥解放战争、建立新中国；从社会主义改造到全面建设社会主义，毛泽东同志的每一个人生节点都与中国革命和建设的伟大历程紧密相连。

### 1.2 项目缘起

2021年12月26日，在毛泽东同志诞辰128周年之际，共产党员网发布了《100张照片回顾毛泽东主席一生中的难忘瞬间》专题，精选了100张珍贵历史照片，按照时间顺序记录了毛泽东主席从1918年求学时期到1965年重上井冈山的人生重要节点。

本项目以此专题内容为核心数据来源，建设一个交互式、沉浸式的纪念网站，通过现代化的 Web 技术手段，以时间轴的形式生动展现毛泽东主席波澜壮阔的一生，让广大人民群众特别是青少年群体能够便捷地了解历史、缅怀伟人。

### 1.3 项目意义

1. **教育意义**：为党史学习、爱国主义教育提供生动的数字化素材。
2. **纪念意义**：以技术手段永久保存和传播珍贵历史影像资料。
3. **传播意义**：通过互联网扩大红色文化的传播覆盖面和影响力。
4. **技术意义**：探索现代 Web 技术在红色文化传播中的应用。

## 2. 项目目标

### 2.1 总体目标

建设一个**视觉庄重、交互流畅、内容详实、适配多端**的毛主席生平纪念网站，以时间轴为主线，以100张珍贵历史照片为载体，完整呈现毛泽东主席一生中的重要时间节点和历史瞬间。

### 2.2 具体目标

| 目标维度 | 具体指标 |
|---------|---------|
| 内容完整性 | 100个时间节点全部上线，每个节点包含照片+文字说明 |
| 用户体验 | 首屏加载时间 ≤ 2秒，时间轴滚动流畅 60fps |
| 多端适配 | 支持 PC、平板、手机三端，覆盖主流浏览器 |
| 访问性能 | 支持 1000 并发用户，日 PV ≥ 10000 |
| 可维护性 | 后台支持内容增删改查，无需修改代码即可更新 |
| 可用性 | 系统可用性 ≥ 99.5% |

### 2.3 非目标（本项目不包含）

1. 不包含毛泽东同志著作的全文展示。
2. 不包含用户注册登录系统（留言功能采用匿名方式）。
3. 不包含电商、支付等商业化功能。
4. 不包含社交分享以外的互动功能。
5. 不包含视频/音频等多媒体内容（本期仅图片+文字）。

## 3. 目标用户

### 3.1 用户画像

| 用户类型 | 占比预估 | 特征描述 | 核心需求 |
|---------|---------|---------|---------|
| 党员干部 | 25% | 各级党政机关、企事业单位党员 | 学习党史、了解伟人生平 |
| 青少年学生 | 30% | 中小学及高校学生 | 爱国主义教育、历史学习 |
| 历史爱好者 | 20% | 对近现代史、党史感兴趣的公众 | 查阅历史资料、观看珍贵照片 |
| 普通网民 | 20% | 通过社交媒体等渠道访问的普通用户 | 浏览了解、缅怀纪念 |
| 研究人员 | 5% | 党史研究、历史教学工作者 | 获取权威资料、引用参考 |

### 3.2 用户场景

**场景一：党员干部党史学习**
张同志是某机关党支部组织委员，在组织主题党日活动时，需要带领支部党员学习毛泽东同志的生平事迹。他打开本网站，通过时间轴功能，按照年代顺序逐一浏览毛主席的重要人生节点，并在每个节点查看详细的历史照片和文字说明，为支部学习提供了丰富的素材。

**场景二：中学生历史课学习**
李同学是高二学生，在学习中国近现代史课程时，需要了解毛泽东同志的革命历程。她通过手机访问本网站，利用移动端友好的时间轴界面，快速定位到"延安时期""解放战争"等关键阶段，通过珍贵的历史照片直观感受历史场景，加深了对课本知识的理解。

**场景三：历史爱好者查阅资料**
王先生是一位退休教师，对党史有浓厚兴趣。他在本网站上通过搜索功能查找"重庆谈判""开国大典"等关键词，快速定位到相关时间节点，并在照片画廊模式下仔细浏览每一张历史照片，还在留言墙上写下了自己的缅怀之情。

## 5. 功能需求

### 5.1 首页模块（P0）

#### FR-1.1 英雄区域展示

**需求描述**：首页顶部展示一个全屏的英雄区域（Hero Section），包含毛主席肖像、网站主标题、副标题和进入时间轴的引导按钮。

**详细要求**：
- 背景：深红色渐变背景（#8B0000 → #2F0000），营造庄重肃穆的氛围
- 主标题：「永远的怀念——毛泽东主席生平纪念」，字体使用思源宋体，字号 48px（PC端）/ 28px（移动端），白色
- 副标题：「100张珍贵照片，重温伟人波澜壮阔的一生」，字号 20px / 16px，浅金色（#D4AF37）
- 肖像：毛主席标准像，居中或偏左展示，带柔和光晕效果
- 引导按钮：「进入时间轴」按钮，点击平滑滚动至时间轴区域
- 装饰元素：底部可添加红旗、松柏等装饰性 SVG 图案
- 动画效果：页面加载时标题淡入，肖像缓慢放大

**验收标准**：
- [ ] 英雄区域占满首屏（100vh）
- [ ] 所有文字在不同屏幕尺寸下清晰可读
- [ ] 「进入时间轴」按钮点击后平滑滚动至时间轴区域
- [ ] 页面加载动画流畅，无闪烁

#### FR-1.2 网站导航栏

**需求描述**：页面顶部固定导航栏，包含网站 Logo、导航菜单和搜索入口。

**详细要求**：
- 导航栏固定在页面顶部（position: fixed），滚动时始终可见
- 背景：半透明深红色（rgba(139, 0, 0, 0.95)），带毛玻璃效果
- Logo：左侧显示网站名称「毛主席纪念」+ 五角星图标
- 导航菜单：首页、时间轴、照片画廊、纪念留言、关于
- 搜索入口：右侧搜索图标，点击展开搜索框
- 滚动行为：向下滚动时导航栏高度缩小（80px → 60px）
- 移动端：导航菜单收起为汉堡菜单

**验收标准**：
- [ ] 导航栏在所有页面固定显示
- [ ] 导航链接点击跳转正确
- [ ] 搜索框展开/收起动画流畅
- [ ] 移动端汉堡菜单功能正常

#### FR-1.3 页脚信息

**需求描述**：页面底部展示版权信息、数据来源声明和友情链接。

**详细要求**：
- 背景：深色（#1a1a1a）
- 内容：版权声明、数据来源（共产党员网）、技术支持信息
- 友情链接：共产党员网、人民网·中国共产党新闻网等
- 备案信息占位（如有）
- 字体颜色：浅灰色（#999）

### 5.2 时间轴模块（P0 - 核心功能）

#### FR-2.1 时间轴主体展示

**需求描述**：以垂直时间轴的形式，按照时间顺序展示毛主席一生中的重要节点。每个节点包含年份、事件标题、缩略图和简要描述。

**详细要求**：

**布局结构**：
- 时间轴采用**左右交替**布局：奇数节点在左侧，偶数节点在右侧
- 中间有一条垂直主线（红色渐变，#8B0000），贯穿整个时间轴
- 每个节点在主线上有一个圆形标记点（金色 #D4AF37），带光晕动画
- 移动端：所有节点统一在左侧，主线在左侧

**节点卡片内容**：
- 年份/日期：大号金色字体，突出显示
- 事件标题：白色字体，2行截断
- 缩略图：圆角矩形，宽度 300px（PC）/ 100%（移动端）
- 简要描述：浅灰色字体，3行截断
- 「查看详情」按钮：点击打开详情弹窗

**滚动动画**：
- 节点进入视口时，从对应方向滑入（左侧节点从左滑入，右侧节点从右滑入）
- 主线标记点在节点进入视口时点亮（放大+发光）
- 使用 Intersection Observer API 实现滚动触发动画
- 动画时长 0.6s，缓动函数 ease-out

**数据加载**：
- 首次加载显示前10个节点
- 滚动到底部时，懒加载后续节点（每次加载10个）
- 加载时显示骨架屏

**验收标准**：
- [ ] 时间轴按时间正序排列（1918年→1965年）
- [ ] 左右交替布局在 PC 端正确显示
- [ ] 移动端单列布局正常
- [ ] 滚动动画流畅，60fps
- [ ] 懒加载功能正常，无重复加载
- [ ] 节点点击可打开详情弹窗

#### FR-2.2 年代快速筛选

**需求描述**：时间轴顶部提供年代筛选标签，用户可快速跳转到特定年代区间。

**详细要求**：

**年代分类**：
| 年代标签 | 时间范围 | 说明 |
|---------|---------|------|
| 求学探索 | 1918—1926 | 湖南求学、建党初期 |
| 革命征程 | 1927—1936 | 秋收起义、井冈山、长征 |
| 延安岁月 | 1937—1945 | 抗日战争、延安整风 |
| 解放战争 | 1946—1949 | 重庆谈判、三大战役、开国大典 |
| 建国初期 | 1950—1959 | 社会主义改造、一五计划 |
| 社会主义建设 | 1960—1965 | 全面建设社会主义时期 |

**交互方式**：
- 年代标签横向排列，默认全部高亮
- 点击某个年代标签，时间轴平滑滚动至该年代的第一个节点
- 被选中的年代标签高亮显示（金色背景）
- 支持多选筛选（按住 Ctrl/Cmd 多选）

**验收标准**：
- [ ] 6个年代标签正确显示
- [ ] 点击标签平滑滚动至对应位置
- [ ] 选中状态视觉反馈清晰
- [ ] 筛选后非该年代的节点淡化显示

#### FR-2.3 时间轴进度指示器

**需求描述**：页面右侧固定一个垂直进度指示器，显示当前浏览位置在整体时间轴中的进度。

**详细要求**：
- 固定在页面右侧中间位置
- 显示当前年份（如"1949年"）
- 进度条显示当前浏览位置百分比
- 点击进度条可快速跳转
- 移动端隐藏

### 5.3 节点详情模块（P0）

#### FR-3.1 详情弹窗展示

**需求描述**：用户点击时间轴上的节点卡片或「查看详情」按钮，弹出该节点的详细信息。

**详细要求**：

**弹窗布局**：
- 模态弹窗，居中显示，宽度 800px（PC）/ 90%（移动端）
- 背景遮罩：半透明黑色（rgba(0,0,0,0.8)），点击关闭
- 弹窗内容区域：深色背景（#1a1a1a），圆角 12px

**弹窗内容**：
1. **顶部**：年份/日期（大号金色字体）+ 关闭按钮
2. **图片区域**：大图展示（最大宽度 100%，最大高度 500px），支持点击放大
3. **图片说明**：图片下方的说明文字
4. **事件描述**：详细的事件背景和意义说明
5. **历史背景**：该节点的历史背景补充信息
6. **导航按钮**：上一个/下一个节点切换按钮

**交互行为**：
- 弹窗打开时背景页面禁止滚动
- 支持 ESC 键关闭弹窗
- 支持左右箭头键切换上一个/下一个节点
- 图片支持点击放大查看（全屏灯箱模式）
- 弹窗内支持鼠标滚轮滚动

**验收标准**：
- [ ] 弹窗正确显示节点所有信息
- [ ] 关闭功能正常（点击遮罩/关闭按钮/ESC键）
- [ ] 上一个/下一个切换功能正常
- [ ] 图片放大灯箱功能正常
- [ ] 键盘快捷键功能正常
- [ ] 移动端弹窗适配正常

#### FR-3.2 图片灯箱模式

**需求描述**：在详情弹窗中点击图片，进入全屏灯箱模式查看大图。

**详细要求**：
- 全屏黑色背景
- 图片居中显示，支持缩放（鼠标滚轮/双指缩放）
- 支持拖动平移查看
- 左右箭头切换同一节点的多张图片
- 显示图片索引（如 1/3）
- ESC 键或点击关闭按钮退出

### 5.5 搜索模块（P1）

#### FR-5.1 全站搜索

**需求描述**：用户可通过关键词搜索时间节点。

**详细要求**：

**验收标准**：

- [ ] 关键词高亮显示
- [ ] 搜索建议实时响应
- [ ] 搜索历史功能正常
- [ ] 空结果提示友好

### 5.6 纪念留言模块（P1）

#### FR-6.1 留言展示

**需求描述**：展示用户缅怀留言，以弹幕或瀑布流形式滚动展示。

**详细要求**：
- 留言展示区域：首页底部或独立页面
- 展示形式：卡片瀑布流，每条留言显示昵称、内容、时间
- 默认显示最新50条留言
- 滚动加载更多
- 管理员审核通过的留言才展示

#### FR-6.2 留言提交

**需求描述**：用户可匿名提交缅怀留言。

**详细要求**：
- 表单字段：昵称（选填，默认"匿名网友"）、留言内容（必填，最多200字）
- 内容安全：提交前进行敏感词过滤
- 提交后提示："留言已提交，审核通过后将展示"
- 频率限制：同一 IP 每小时最多提交5条
- 验证码：可选启用图形验证码防刷

**验收标准**：
- [ ] 留言提交功能正常
- [ ] 敏感词过滤有效
- [ ] 频率限制生效
- [ ] 提交提示友好

### 5.7 后台管理模块（P1）

#### FR-7.1 管理员登录

**需求描述**：管理员通过账号密码登录后台管理系统。

**详细要求**：
- 登录页面：用户名 + 密码 + 图形验证码
- 密码加密：bcrypt 存储
- 登录态：JWT Token，有效期 24 小时
- 安全：5次密码错误锁定账户 30 分钟
- 登出：清除 Token

#### FR-7.2 时间节点管理

**需求描述**：管理员可对时间节点进行增删改查操作。

**详细要求**：
- 列表页：分页展示所有节点，支持搜索、筛选、排序
- 新增：填写日期、标题、描述、图片URL、历史背景等字段
- 编辑：修改节点信息
- 删除：软删除（标记 is_deleted）
- 批量操作：批量删除、批量修改年代分类
- 排序：支持手动调整节点顺序

#### FR-7.3 留言审核管理

**需求描述**：管理员审核用户提交的留言。

**详细要求**：
- 待审核列表：展示所有待审核留言
- 审核操作：通过/拒绝/删除
- 批量审核：批量通过/拒绝
- 已审核列表：查看已通过/已拒绝的留言
- 敏感词管理：维护敏感词库

#### FR-7.4 数据统计面板

**需求描述**：展示网站访问统计数据。

**详细要求**：
- 今日访问量、总访问量
- 时间节点浏览排行 Top 10
- 留言统计：今日新增、待审核数
- 访问趋势图：近30天访问量折线图
- 设备分布：PC/移动端占比饼图

| API 响应时间 | ≤ 500 毫秒（P95） |
| 图片加载时间 | 单张 ≤ 1 秒（CDN加速） |
| 并发用户数 | 支持 1000 并发 |

**浏览器兼容**：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- 移动端：iOS Safari 14+、Chrome for Android 90+

**设备兼容**：
- PC：分辨率 ≥ 1280×720

### 6.3 安全需求

4. **敏感词过滤**：留言内容进行敏感词过滤
5. **频率限制**：API 接口启用速率限制
6. **HTTPS**：全站强制 HTTPS
7. **密码安全**：bcrypt 加密存储，不可逆
8. **文件上传**：图片上传校验类型和大小

1. 系统可用性 ≥ 99.5%
2. 支持 7×24 小时访问
3. 数据每日备份，保留 30 天

2. 核心逻辑单元测试覆盖率 ≥ 70%

4. 前后端分离，接口契约清晰
5. 配置项集中管理，支持环境变量
