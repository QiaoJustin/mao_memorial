## 一、项目简介

本项目旨在建设一个以**时间轴**为主线、以**珍贵历史照片**为核心载体的毛主席生平纪念网站。网站按照毛主席一生从小到大的时间顺序，展示其生命中最重要的时间节点和重大事件，每个时间节点配以真实历史照片和详实的文字说明，以此缅怀伟大领袖毛泽东主席。

项目数据来源：共产党员网《100张照片回顾毛泽东主席一生中的难忘瞬间》
- 原文链接：https://www.12371.cn/2021/12/26/ARTI1640485049308845.shtml
