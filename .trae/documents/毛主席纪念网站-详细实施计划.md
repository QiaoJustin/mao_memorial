## 一、摘要 (Summary)

本计划将毛主席生平纪念网站从零代码状态分 **9 个阶段** 逐步实施完成。项目为 Next.js 14+ 全栈应用，包含前台展示系统（首页、时间轴、照片画廊、搜索、留言墙）和后台管理系统（节点CRUD、留言审核、数据统计）。

**关键技术决策**：
- 数据库环境：Docker Compose 一键启动 MySQL 8.0 + Redis 7
- 图片处理：下载 100 张照片到本地 `public/images/timeline/`，使用 Next.js Image 优化
- 渲染策略：首页/时间轴 SSG，详情弹窗/后台 CSR
- UI 组件：前台 shadcn/ui，后台 Ant Design 5+

**预计阶段**：9 个阶段，每个阶段有明确的产出物和验证标准。

**任务清单**：

   - 创建 `docker-compose.yml`（MySQL 8.0 + Redis 7-alpine，按 docs/08 §3.3）

### 阶段三：后端基础工具与公共 API

**目标**：实现后端工具库与公开 API 路由（不含 admin）。

**任务清单**：

1. **实现核心工具库**（`src/lib/`）
   - `api-response.ts`：统一响应封装 `apiSuccess()` / `apiError()` / `apiPaginate()`，格式按 docs/06 §1.3
   - `validate.ts`：Zod 参数校验工具 `validateQuery()` / `validateBody()`
   - `cache.ts`：Redis 缓存封装 `cacheGet()` / `cacheSet()` / `cacheInvalidate()`
   - `logger.ts`：Winston 日志工具
   - `error.ts`：自定义错误类 `ApiError`，统一错误处理
   - `auth.ts`：JWT 签发/验证工具，`signToken()` / `verifyToken()` / `getAuthUser()`
   - `rate-limit.ts`：基于 Redis 的频率限制 `rateLimit(key, windowMs, max)`
   - `sensitive.ts`：敏感词 Trie 树过滤 `filterSensitive(content)`
   - `upload.ts`：文件上传处理（multipart 解析）
   - `image.ts`：sharp 图片处理（缩略图生成）

2. **创建类型定义**（`src/types/`）
   - `api.ts`：ApiResponse、PaginatedResponse、ErrorDetail
   - `timeline.ts`：TimelineNode、TimelineNodeDetail、Photo、Era
   - `message.ts`：Message、MessageStatus
   - `admin.ts`：Admin、AdminRole、AdminStatus
   - `index.ts`：统一导出

3. **创建常量定义**（`src/constants/`）
   - `eras.ts`：6 个年代常量（id/name/period/color）
   - `cache-keys.ts`：缓存 Key 模板（`timeline:page:{page}:size:{size}` 等）
   - `error-codes.ts`：错误码枚举
   - `config.ts`：配置常量（分页默认值、JWT 过期时间等）

4. **实现公共 API 路由**（`src/app/api/v1/`）
   - `health/route.ts`：GET 健康检查（检查 DB + Redis 连接）
   - `settings/route.ts`：GET 公开系统设置（is_public=true 的设置项）
   - `eras/route.ts`：GET 年代列表（Redis 缓存 1 小时）

5. **实现时间轴 API**（`src/app/api/v1/timeline/`）
   - `route.ts`：GET 时间轴列表（分页、年代/年份筛选，Redis 缓存 5 分钟）
   - `[id]/route.ts`：GET 节点详情（含 photos 和 tags）
   - `[id]/adjacent/route.ts`：GET 相邻节点（prev/next）
   - `[id]/view/route.ts`：POST 记录浏览量（Redis SET 去重，IP 维度 1 小时）
   - `featured/route.ts`：GET 精选节点（limit 参数，默认 5）

6. **实现照片 API**（`src/app/api/v1/photos/`）
   - `route.ts`：GET 照片画廊（分页、年代/年份/节点筛选）
   - `[id]/route.ts`：GET 照片详情（含关联节点信息）

7. **实现搜索 API**（`src/app/api/v1/search/`）
   - `route.ts`：GET 搜索（MySQL FULLTEXT 索引，ngram 分词，高亮 `<mark>` 标签）
   - `suggest/route.ts`：GET 搜索建议（前缀匹配，Redis 缓存 10 分钟）
   - `hot/route.ts`：GET 热门搜索（Redis ZSET 排序）

8. **实现留言 API**（`src/app/api/v1/messages/`）
   - `route.ts`：GET 留言列表（仅 approved，分页）+ POST 提交留言（敏感词过滤 + Redis 频率限制 IP/1小时/5条）
   - `[id]/like/route.ts`：POST 点赞（Redis SET 去重，IP+留言ID 维度）

9. **实现认证 API**（`src/app/api/v1/auth/`）
   - `login/route.ts`：POST 登录（bcrypt 验证 + JWT 签发 + HttpOnly Cookie；5 次错误锁定 30 分钟）
   - `logout/route.ts`：POST 退出（清除 Cookie）
   - `me/route.ts`：GET 当前用户信息
   - `password/route.ts`：PUT 修改密码

10. **配置中间件**
    - `src/middleware.ts`：拦截 `/admin/*`（排除 `/admin/login`），验证 JWT Token，无效则重定向到登录页

**验证标准**：
- [ ] `GET /api/v1/health` 返回 200，数据库和 Redis 状态为 connected
- [ ] `GET /api/v1/eras` 返回 6 个年代
- [ ] `GET /api/v1/timeline?page=1&pageSize=10` 返回 10 个节点（按 date_sort 升序）
- [ ] `GET /api/v1/timeline/48` 返回开国大典节点详情，含 photos 数组
- [ ] `GET /api/v1/timeline/48/adjacent` 返回 prev 和 next
- [ ] `GET /api/v1/search?q=延安` 返回包含延安的节点，关键词高亮
- [ ] `POST /api/v1/messages` 提交留言返回 201，状态 pending
- [ ] `POST /api/v1/auth/login` 用 admin/admin123 登录返回 token
- [ ] 未登录访问 `/admin/dashboard` 被重定向到 `/admin/login`

**产出物**：完整的公开 API（不含 admin 管理接口）+ 工具库 + 中间件
